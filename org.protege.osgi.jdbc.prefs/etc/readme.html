OSGi JDBC Driver Preferences Plugin

This plugin is intended to give a common framework of support for Protege 4 plugins that will use jdbc to interact with sql. This plugin allows users to choose jdbc drivers that will be used by Protege plugins using jdbc. This plugin plays a support role and will only be useful if required by some other plugin.
